# GitHub

gh CLIを使用したGitHub操作の包括的なガイド。タスクに応じて適切なリファレンスを参照する。

## タスク別リファレンス

| タスク | リファレンス |
|--------|-------------|
| PR作成・レビュー・マージ | [references/pr.md](references/pr.md) |
| Issue作成・管理・検索 | [references/issues.md](references/issues.md) |
| ワークフロー確認・実行・ログ | [references/actions.md](references/actions.md) |
| リリース作成・タグ管理 | [references/release.md](references/release.md) |

### 各領域の概要

- **Pull Request** - PR作成、レビュー（承認/変更リクエスト）、コメント、マージ、CI確認、編集
- **Issues** - Issue作成、一覧・検索、コメント、編集、クローズ/リオープン、ラベル・マイルストーン管理
- **Actions** - ワークフロー一覧・実行、ログ確認、ジョブ再実行、アーティファクト、キャッシュ管理
- **Release** - リリース作成（自動ノート生成）、アセットアップロード、編集・削除、セマンティックバージョニング
